HTMLify Developer API

Integrate HTMLify into your own apps with our simple REST API.

Authentication

No Auth Required

Most public endpoints are accessible without any API key. Rate limits apply to unauthenticated requests.

Endpoints
GET /api/v1/file/{id}

Fetch details about a specific file including its metadata and content preview.

curl {{ request.url_root }}api/v1/file/123
Parameter Type Description
id String The unique identifier of the file.
GET /api/v1/search

Search for public files by query or username.

curl {{ request.url_root }}api/v1/search?q=cool-project
Parameter Type Description
q String Search query string.
user String Filter by username.
Best Practices

Rate Limiting
